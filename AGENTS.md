# PASSIO ARCHITECTURAL & DESIGN DIRECTIVES

All development work on Passio MUST strictly adhere to the **Passio Design Bible** (`/PASSIO_DESIGN_BIBLE.md`).

## Core Directives:
1. **Design Motto**: "Az ekran. Az renk. Az hareket. Çok düşünce." (Fewer screens. Fewer colors. Less movement. More thought.)
2. **Three Principles**:
   - **Sessizlik (Silence)**: Calm typography, no loud buttons, smooth purposeful micro-animations, generous whitespace.
   - **Odak (Focus)**: Single-purpose dedicated screens. Never turn Passio into a multi-panel dashboard or feature salad.
   - **Süreklilik (Continuity)**: Offline-first data persistence, seamless passage between reading, thinking, and writing.
3. **The Golden Rule**: Before adding any feature, ask: *"Does this feature make Passio deeper, or just bigger?"* If it only makes it bigger, do NOT add it.
