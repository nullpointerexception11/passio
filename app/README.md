# PASSIO UYGULAMA KAYNAK KODLARI (`/app` -> `/src`)

Passio istemci kaynak kodları ve uygulama bileşenleri bu dizinde / `src` altında yer almaktadır.

## Dizin Haritası
- `src/core/`: İş mantığı, veri modelleri ve servisler.
- `src/components/`: Atomic UI bileşenleri (Atoms, Molecules, Organisms).
- `src/db/`: Şemalar ve yerel veritabanı sürücüleri.
- `src/data/`: Örnek PDF ve kütüphane içerikleri.
